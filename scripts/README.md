# Scripts de diagnostic

Ce dossier contient des scripts utiles pour diagnostiquer et tester le système.

## Scripts disponibles

### `debug-cookie.js`
Vérifie l'état des cookies configurés pour chaque équipe.
```bash
node scripts/debug-cookie.js
```

### `check-db.js`
Affiche les statistiques de la base de données (équipes, utilisateurs, configurations).
```bash
node scripts/check-db.js
```

### `test-cookie-save.js`
Teste le chiffrement/déchiffrement des cookies.
```bash
node scripts/test-cookie-save.js
```
